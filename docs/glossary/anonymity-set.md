---
id: anonymity-set
title: Anonymity set
summary: The number of plausible owners a transaction or output could belong to; larger sets provide stronger privacy.
related: [utxo-segregation-playbook, utxo-selection-heuristics]
keywords: [privacy, heuristics]
---

Privacy improves when an observer cannot easily distinguish your activity from many others. Good input selection, avoiding address reuse, and using uniform script types help maintain larger anonymity sets.
