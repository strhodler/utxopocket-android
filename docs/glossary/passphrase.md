---
id: passphrase
title: Passphrase
summary: An optional secret added to the seed that derives a different wallet; protects against seed disclosure.
related: [self-custody-hygiene]
keywords: [backup, security]
---

A passphrase extends the seed to create a distinct wallet. It must be stored separately from the seed phrase. Loss or error renders funds inaccessible, so practice recovery with test funds.
