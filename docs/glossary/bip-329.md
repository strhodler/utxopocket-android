---
id: bip-329
title: BIP‑329 labels
summary: A standard JSON format for exporting and importing wallet labels across applications.
related: [provenance-labeling]
keywords: [labels, interoperability]
---

BIP‑329 defines a common schema to exchange labeling metadata (address, output, transaction notes) between wallets. Using it preserves provenance during migrations and keeps spending policies consistent.
