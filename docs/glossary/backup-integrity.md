title: Backup integrity
summary: Confidence that a backup file is authentic, complete, and untampered before import.
related: [encrypted-watch-only-backup, backup-recovery-drill, descriptor-maps-and-recovery]
keywords: [integrity, validation, backup]
---

Backup integrity means the file decrypts with the correct passphrase, passes schema and security validation, and previews expected wallet metadata before import.
