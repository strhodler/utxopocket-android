title: Tor
summary: A network that routes traffic through relays to hide origin and destination, reducing metadata leaks.
related: [electrum-servers, self-custody-hygiene]
keywords: [privacy, networking]

Using Tor with wallets and servers hides IP metadata from third parties. Prefer Tor for Electrum queries and peer connections so address lookups and broadcasts are unlinkable to your network identity.
