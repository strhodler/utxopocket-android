title: Dust
summary: An output whose value is near or below the cost to spend, often discouraged by standardness rules.
related: [fee-selection-playbook, change-output-hygiene]
keywords: [dust, fees, standardness]

Creating dust leaks patterns and increases future fees. Prefer avoiding dust outputs in drafts and consolidate tiny UTXOs during low‑fee windows when it won’t link compartments.
