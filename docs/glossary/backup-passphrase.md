---
id: backup-passphrase
title: Backup passphrase
summary: The secret phrase required to decrypt and import an encrypted .ubak backup.
related: [encrypted-watch-only-backup, encrypted-backup, backup-recovery-drill]
keywords: [passphrase, backup, recovery]
---

A backup passphrase encrypts and unlocks `.ubak` backups. If it is lost, the backup file is unusable, so store the passphrase separately from the backup file.
