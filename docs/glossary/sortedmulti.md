---
id: sortedmulti
title: sortedmulti(...)
summary: Descriptor function that enforces lexicographic public key order for multisig to improve interoperability.
related: [descriptors-advanced]
keywords: [descriptor, multisig, sortedmulti]
---

Using `sortedmulti` avoids ambiguity from key order differences across tools. It ensures the resulting script and addresses are consistent among wallets that follow descriptor standards.
