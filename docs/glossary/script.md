id: script
title: Script type
summary: The spend template used by an address (e.g., P2PKH, P2WPKH, P2TR) with implications for fees and privacy.
related: [script-type-tradeoffs, address-format-fingerprints]
keywords: [p2pkh, p2wpkh, taproot]
---

Script type defines how spending conditions are encoded and validated. Different templates have distinct sizes, fees, and fingerprints. Avoid mixing multiple script types in a single spend to reduce clustering risk.
