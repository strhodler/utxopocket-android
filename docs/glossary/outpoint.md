---
id: outpoint
title: Outpoint
summary: A specific UTXO reference identified by `txid:vout`, used as an input in a transaction.
related: [utxo-selection-heuristics]
keywords: [utxo, input, txid]
---

Wallets select outpoints as inputs when drafting spends. Keeping compartments separate and avoiding merges reduces what outpoints reveal about ownership.
