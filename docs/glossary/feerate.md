---
id: feerate
title: Feerate (sats/vB)
summary: The fee density per virtual byte that miners prioritize; central to estimating confirmation time.
related: [fee-selection-playbook]
keywords: [fees, mempool]
---

Feerate is the amount paid per virtual byte of transaction data, expressed in sats/vB. Higher feerates compete more effectively for limited block space. Select a target band based on urgency and mempool conditions, not only on absolute fee totals.
