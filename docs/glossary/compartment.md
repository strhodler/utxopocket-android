---
id: compartment
title: Compartment
summary: A labeled purpose‑bucket for UTXOs that should not be merged with others to preserve privacy.
related: [utxo-segregation-playbook, change-output-hygiene]
keywords: [segregation, labels, privacy]
---

Compartmentalizing coins by purpose (e.g., savings, donations) reduces linkability. Respect compartments in input selection and change policy; use BIP‑329 labels to record intent.
